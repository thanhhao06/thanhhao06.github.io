## Run Locally

Clone the repository:

```bash
git clone https://github.com/thanhhao06/thanhhao06.github.io.git
cd thanhhao06.github.io
```

### Python 3

```bash
python -m http.server 8000
```

Or on Linux:

```bash
python3 -m http.server 8000
```

Then open:

```txt
http://localhost:8000
```

## Deployment

This website is deployed using **GitHub Pages**.

Because the repository name is:

```txt

```

GitHub Pages can publish it directly as:

```txt
https://thanhhao06.github.io
```
